# Driving-Retail-Profitability-through-Conversion-and-Inventory-Optimization-using-SQL-and-Excel
Analyzed store sales, footfall, and inventory data using SQL and Excel to improve conversion and stock efficiency. Identified low-performing SKUs, shrink issues, and poor replenishment patterns. Recommended actions that improved sell-through, reduced excess stock, and boosted overall retail profitability.
